# 티스토리 백링크 글 관리

## 목적
tistory.com(DA 높음)에서 `m.blog.naver.com` 링크를 제공하여 구글봇이 네이버 블로그를 발견하게 유도.

## 파일 규칙
- 파일명: `{번호}-{카테고리}-{순번}.md` (예: `01-맛집-1.md`)
- 한 파일당 포스트 링크 10개 이내
- 10개 초과 시 순번 올려서 새 파일 (예: `01-맛집-5.md`)

## 글 작성 규칙
- 제목: `# 자연스러운 제목`
- 본문: 각 포스트를 `[제목](m.blog.naver.com URL)` + 한두 줄 설명
- 링크는 반드시 `https://m.blog.naver.com/` 형태
- 이미지, 태그 불필요

## 새 포스트 추가 시
1. 해당 카테고리 파일에 링크 추가
2. 10개 넘으면 새 파일로 분리
3. 티스토리 글 수정 반영
